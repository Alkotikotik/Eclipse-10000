Format of 32bit instructions

000000     | 00000 | 00000    | 0000000000000000
opcode (6b)| rx0(5b)   |rx1(spare)| (spare/address)
rx0 = rx0 (operation) rx1


ALU instructions:
000001 : Add 
000011 : Sub
000111 : LOMUL
001101 : HIMUL
000010 : XOR
000110 : OR
001110 : AND
001111 : NOT
001000 : SHL
001100 : SHR
001010 : SRA

Memory 
010001 : LOAD //load instant 
100011 : LDR
100111 : STR 

Branching 

110000 : BEQ 
110001 : BNE 
110011 : BS
110111 : BG 
111111 : JMP
